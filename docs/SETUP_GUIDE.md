# **Ranger Med-Core -- Setup Guide**

## **Backend Setup**
1. Navigate to backend folder
```sh
cd backend
```

2. Install dependencies
```sh
npm install
```

3. Copy `.env.example` to `.env` and fill keys

4. Run backend
```sh
npm run dev
```

## **Frontend Setup**
1. Navigate to frontend folder
```sh
cd frontend
```

2. Install dependencies
```sh
npm install
```

3. Copy `.env.example` to `.env` and fill keys

4. Run frontend
```sh
npm run dev
```

## Notes
- Backend runs on port 5000, frontend on port 3000
- Make sure MongoDB is running locally
- Use Postman or VS Code (Thunder Client) or Frontent to test APIs
